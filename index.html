冬月

发表于 2025-12-22 更新于 2025-12-27 分类于 感悟

#冬至
昨天是冬至，冬至过，地冻破。天气在这一天才开始真正的冷了，开始数九了，一九第一天。今天早上起来，气温明显就降了，今明两天还有雨雪，雨雪后要冷几天了。
我们这边冬至也要吃饺子，俗话说，冬至不端饺子碗，冻掉耳朵没人管。正好妹妹一家都在这里，几个孩子都要吃白菜馅的饺子。下午去地窖扒出来两棵白菜，超市搅了一点肉馅，开始动工了。冬至大如年，人间小团圆，这也算吃了个团圆饭啊。
#雪
上一周，一场期待己久的雪终于下下来了，今年的第一场雪来的正是时候。虽然一晚上下的都很大，但是地面基本没有雪，毕竟第一场雪，地面还没有冷下来。只有树上，车上，和院子里的盆盆罐罐存了一点雪。也算是到处白茫茫一片了。
明天是雨雪天气，希望再下一场雪吧。雪天然有净化空气的作用，下一场雪，把病毒细菌灰尘压一压，净化一下，也把最近的各种传染病压一下，安安稳稳过个新年吧。

更新手机

发表于 2025-12-08 更新于 2025-12-27 分类于 文章

最近想了又想，终于决定还是把手机更新了iOS26。之前用的iOS18，感觉还挺不错的，架不住各种自媒体的介绍和宣传，一开始看着难看的26，居然越看越顺眼，想换了试试。
顺便把手表也更新了，苹果手机不更新，手表也更新不了。除了玻璃效果，其他没什么变化，也没有卡顿的现象。手表更新完稍微有点卡顿，到过了一会就好了。
我还发现了一个新的功能，现在垃圾电话太多，我把语音信箱留言翻出来了。现在无论任何陌生电话都直接转成语音留言了，通讯录的电话可以直接接，简直太爽了。话说最近的骚扰电话太多了，拦截软件都拦不住，每天换着电话号码给你打。
小米手机也有类似功能，但解决方式是小爱同学和对面聊天，聊的有来有回的，实在有点拖沓，感觉还是苹果这个直截了当。

立冬

发表于 2025-11-07 更新于 2025-12-27 分类于 感悟

今天立冬，天气和节气很配合，早上起来还晴着天，没一会就阴天了，一整天都阴沉沉的。下午放学时，居然开始下雨了，一直下到现在也没停。
立冬不端饺子碗，冻掉耳朵没人管。本来今天晚上应该吃饺子的，但是上周就跟孩子约好的，要在外面吃，要去吃米线。所以放了学就直奔米线店。孩子对面条和米线真是没有一点抵抗力，吃多少次都吃不腻，我不行。只给她们点好了餐，我没吃。偶尔吃一次可以，经常吃我就没兴趣了，还是不如家里的饭好吃。
立冬是冬季的开始，也是万物积蓄力量的开始。前两天走过河边，河里的莲藕开始干枯，荷叶枯了也变黑了，荷叶杆也断了，耷拉着脑袋。但是我知道，莲藕马上快出淤泥了。门口的银杏树，叶子也从黄绿相间，浅黄，现在整个的黄透了，黄色的树叶倒是好看的不得了。只是果实掉了一地，有点异味。但是可以吃到银杏果，有点苦涩的底味。这场雨过后，气温会继续下降，下周末就能到0度了。还没怎么感受秋天的气息，秋已经又到头了。
回顾一下再展望一下，普通人的一年又快到头了。都说年龄越大，时间过得越快。感觉自己年龄不大，但却深深感受到了时间的力量。身体的小毛病多了，想事情也变得直截了当，不想那么多了，没有了很多顾忌。人生就是这样吧，可能年龄大了，也就能更多的为自己活了，不用顾忌那么多别人的看法了。人本来就是自私的动物。
今日立冬，愿你有暖阳可依，亦能有心舟可栖。
今天新闻才出来，前天福建舰入列了，记录一下。
